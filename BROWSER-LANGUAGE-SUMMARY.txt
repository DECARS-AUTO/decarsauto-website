│  🔧 技术实现                                                           │
└───────────────────────────────────────────────────────────────────────┘

  1️⃣ 浏览器语言检测
     const browserLang = navigator.language.toLowerCase();
     
     映射规则:
     • zh-CN, zh-TW, zh-* → 中文 (ZH)
     • de-DE, de-AT, de-* → 德语 (DE)
     • en-US, en-GB, 其他 → 英语 (EN)

  2️⃣ 模式管理
     • auto: 跟随浏览器语言
     • manual: 使用用户选择

  3️⃣ 长按检测
     • mousedown 开始计时
     • 1秒后触发自动模式
     • mouseup/mouseleave 取消

  4️⃣ LocalStorage 数据
     • languagePreference: 'auto' / 'manual'
     • language: 'en' / 'de' / 'zh'

│  🔍 调试命令                                                           │
└───────────────────────────────────────────────────────────────────────┘

  在浏览器控制台运行：

  # 查看当前状态
  console.log('Browser Language:', navigator.language);
  console.log('Detected Language:', detectBrowserLanguage());
  console.log('Language Preference:', 
    localStorage.getItem('languagePreference'));
  console.log('Current Language:', 
    localStorage.getItem('language'));

  # 强制自动模式
  localStorage.setItem('languagePreference', 'auto');
  location.reload();

  # 强制手动中文
  localStorage.setItem('languagePreference', 'manual');
  localStorage.setItem('language', 'zh');
  location.reload();

  # 清除所有设置
  localStorage.removeItem('languagePreference');
  localStorage.removeItem('language');
  location.reload();

┌───────────────────────────────────────────────────────────────────────┐
│  🐛 常见问题                                                           │
└───────────────────────────────────────────────────────────────────────┘

  Q1: 为什么长按没反应？
  A1: 需要按住至少1秒，看到通知即成功

  Q2: 语言未自动检测？
  A2: 检查浏览器语言设置，确认映射规则

  Q3: 手动切换后不能回到自动？
  A3: 长按语言切换器按钮1秒切换到自动模式

  Q4: 通知不显示？
  A4: 清除浏览器缓存，确保CSS已更新

  Q5: 不同设备语言不同步？
  A5: 这是正常的，每个设备独立保存设置

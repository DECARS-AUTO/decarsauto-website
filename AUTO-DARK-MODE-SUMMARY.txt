  ✨ 新增功能：深色模式自动检测系统主题偏好
  
  🤖 自动模式（默认）
     • 检测系统深色模式设置

  🎛️ 手动模式

     • 保存用户选择
     • 不受系统影响

┌───────────────────────────────────────────────────────────────────────┐
│  🔧 技术实现                                                           │
└───────────────────────────────────────────────────────────────────────┘

  1️⃣ 系统主题检测
     const prefersDarkScheme = window.matchMedia(
       '(prefers-color-scheme: dark)'
     );

  2️⃣ 实时监听系统变化
     prefersDarkScheme.addEventListener('change', (e) => {
       if (isAutoMode) {
         applyDarkMode(e.matches);
       }
     });

  3️⃣ 长按检测
     • mousedown 开始计时
     • 1秒后触发自动模式
     • mouseup/mouseleave 取消

  4️⃣ LocalStorage 数据
     • darkModePreference: 'auto' / 'manual'
     • darkMode: 'enabled' / 'disabled'

┌───────────────────────────────────────────────────────────────────────┐
│  🔍 调试命令                                                           │
└───────────────────────────────────────────────────────────────────────┘

  在浏览器控制台运行:

  # 查看当前状态
  console.log('Preference:', 
    localStorage.getItem('darkModePreference'));
  console.log('Dark Mode:', 
    localStorage.getItem('darkMode'));
  console.log('System Prefers Dark:', 
    window.matchMedia('(prefers-color-scheme: dark)').matches);

  # 强制自动模式
  localStorage.setItem('darkModePreference', 'auto');
  location.reload();

  # 强制手动深色
  localStorage.setItem('darkModePreference', 'manual');
  localStorage.setItem('darkMode', 'enabled');
  location.reload();

  # 清除所有设置
  localStorage.removeItem('darkModePreference');
  localStorage.removeItem('darkMode');
  location.reload();

┌───────────────────────────────────────────────────────────────────────┐
│  🐛 常见问题                                                           │
└───────────────────────────────────────────────────────────────────────┘

  Q1: 为什么长按没反应？
  A1: 需要按住至少1秒，看到通知即成功

  Q2: 自动模式不工作？
  A2: 检查浏览器版本，需支持 prefers-color-scheme

  Q3: 系统切换后网站不变？
  A3: 检查是否在手动模式，长按图标切换到自动模式

  Q4: 通知不显示？
  A4: 清除浏览器缓存，确保CSS已更新

  Q5: 刷新后设置丢失？
  A5: 检查LocalStorage是否被禁用或清除

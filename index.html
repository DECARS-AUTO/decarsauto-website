VOLKSWAGEN AG 在线编程项目：

  • CP（Component Protection / 组件保护）：按授权流程执行
  • IMMO（Immobilizer / 防启动系统）：匹配/适配/钥匙相关编程（按授权范围）
  • SVM（Software Version Management / 软件版本管理）：在线软件与参数管理（以车型支持为准）
  • SFD / SFD2（Secure Feature Diagnostic / 安全诊断访问）：引导式授权访问流程（以系统要求为准）
  • VKMS（VAG Key Management System / 钥匙管理系统）：钥匙订购、授权与状态管理（以车型与授权范围为准）
  • FoD / FOD（Feature on Demand / 按需功能）：功能订购、下发与激活（以车辆配置与授权范围为准）

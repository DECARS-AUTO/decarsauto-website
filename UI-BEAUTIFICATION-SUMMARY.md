- 应用于所有卡片元素

## 📊 性能优化

### CSS 优化
- 使用 `transform` 代替 `top/left`
- `will-change` 提示（必要时）
- GPU 加速动画
- 合理使用 `overflow: hidden`

### JavaScript 优化
- Intersection Observer（高效滚动监听）
- 节流滚动事件
- 仅动画化可见元素
- 一次性动画（动画后取消观察）

## 🌐 浏览器兼容性

### 完全支持
- ✅ Chrome 76+
- ✅ Firefox 67+
- ✅ Safari 12.1+
- ✅ Edge 79+

### 优雅降级
- 不支持 `backdrop-filter` 的浏览器显示纯色背景
- Intersection Observer 不支持时跳过动画
- CSS 变量不支持时使用回退值

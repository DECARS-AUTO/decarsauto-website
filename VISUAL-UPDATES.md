# 视觉更新说明 - Visual Updates

## 📋 更新概述

已对网站的视觉效果进行全面优化，增强背景图片与内容卡片之间的协调性。

## ✨ 主要改进

### 1. **玻璃态效果 (Glass-morphism)**
所有内容卡片现在采用现代化的毛玻璃效果：

- ✅ **半透明背景** - 85% 不透明度（白色/深色）
- ✅ **毛玻璃模糊** - `backdrop-filter: blur(10px)`
- ✅ **背景图片透过** - 可以看到背景图片透过卡片
- ✅ **悬停增强** - 悬停时不透明度提升到 95%

### 2. **背景图片透明度统一**
所有部分的背景遮罩统一调整为 **72% 不透明度**：

- 🎨 **核心服务 (Services)** - 从 97% → 72%
- 🎨 **远程服务流程 (Workflow)** - 从 95% → 72%
- 🎨 **信任保障 (Trust)** - 从 97% → 72%
- 🎨 **联系我们 (Contact)** - 保持 72%
- ✅ **与 Hero 区域保持一致**

### 3. **文字对比度优化**

#### 标题优化：
- **颜色变化**: 从橙色主题色 → 深色文字 (`#1a1a1a`)
- **字重增强**: 从 500 → 600（中等 → 半粗）
- **提升可读性**: 在半透明背景上更清晰

#### 编号/标签优化：
- **颜色**: 改为主题橙色 (`var(--primary-color)`)
- **字重**: 增强到 600
- **透明度**: 80% 以保持层次感

#### 描述文字：
- **颜色**: 更深的灰色 (`#4a4a4a`)
- **在浅色半透明背景上有更好的对比度**

### 4. **阴影系统升级**

#### 默认状态：
```css
box-shadow: 0 4px 15px rgba(0, 0, 0, 0.08);
```
- 柔和的灰色阴影
- 轻微深度感

#### 悬停状态：
```css
box-shadow: 0 8px 25px rgba(255, 107, 0, 0.15), 
            0 4px 15px rgba(0, 0, 0, 0.1);
```
- 双层阴影效果
- 橙色主题光晕 + 灰色深度
- 提升视觉层次

### 5. **深色模式完整适配**

所有新增的玻璃态卡片都完全支持深色模式：

- 🌙 **背景色**: `rgba(40, 40, 40, 0.85)` - 深灰色半透明
- 🌙 **边框**: `rgba(80, 80, 80, 0.6)` - 更亮的灰色边框
- 🌙 **文字**: 浅色文字以保持对比度
- 🌙 **悬停效果**: 同样的动画和交互

## 🔧 技术细节

### CSS 特性使用：
```css
/* 玻璃态效果 */
background: rgba(255, 255, 255, 0.85);
backdrop-filter: blur(10px);
-webkit-backdrop-filter: blur(10px);

/* 双层阴影 */
box-shadow: 
  0 8px 25px rgba(255, 107, 0, 0.15),  /* 橙色光晕 */
  0 4px 15px rgba(0, 0, 0, 0.1);        /* 深度阴影 */

/* 流畅过渡 */
transition: var(--transition);  /* 0.3s ease */
```

## 🔄 如何回滚

如果需要恢复到之前的纯色卡片样式，可以：

```css
/* 移除玻璃态效果 */
.service-card,
.workflow-step,
.trust-item,
.contact-card {
    background: var(--bg-white);  /* 纯白背景 */
    backdrop-filter: none;         /* 移除模糊 */
}

/* 恢复原始透明度 */
.services.with-background {
    background-image: linear-gradient(
        rgba(255, 255, 255, 0.97),  /* 97% 不透明 */
        rgba(255, 255, 255, 0.97)
    ), url('images/services-bg.png');
}
```

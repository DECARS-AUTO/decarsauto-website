📝 使用说明

🎯 方式 1: 手动切换
点击语言切换器 → 选择语言 → 进入手动模式
🎯 方式 2: 自动模式
长按（1秒）语言切换器按钮 → 显示通知 → 进入自动模式

工作模式说明：

  • 自动模式（默认）: 跟随浏览器语言设置
  • 手动模式: 用户手动选择，不受浏览器影响

语言映射规则：

  • zh-CN, zh-TW, zh-* → 中文 (ZH)
  • de-DE, de-AT, de-* → 德语 (DE)
  • en-US, en-GB, 其他 → 英语 (EN)

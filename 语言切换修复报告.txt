如果语言切换仍然不工作:

1. 清除浏览器缓存
   ────────────────────────────────────────────────────────────────
   Ctrl + Shift + Delete → 选择"缓存的图片和文件" → 清除数据

2. 检查浏览器控制台
   ────────────────────────────────────────────────────────────────
   F12 → Console → 查找红色错误

3. 验证文件版本
   ────────────────────────────────────────────────────────────────
   确保使用的是最新版本的:
   • index.html (hero-title更新)
   • script.js (翻译数据更新)
   • enhancements.js (动画修复)
   • styles.css (CSS强制显示)

4. 使用无痕模式测试
   ────────────────────────────────────────────────────────────────
   Ctrl + Shift + N → 打开 index.html

如果logo仍然不显示:

1. 检查文件存在
   ────────────────────────────────────────────────────────────────
   确认 images/brands/ 文件夹包含7个logo文件

2. 查看Network标签
   ────────────────────────────────────────────────────────────────
   F12 → Network → 刷新 → 搜索"brands"
   所有logo应该是 200 状态

3. 查看Console日志
   ────────────────────────────────────────────────────────────────
   应该看到 "✅ logo loaded" 消息

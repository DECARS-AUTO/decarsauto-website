Q1: 清除缓存后仍然看到旧版本？
─────────────────────────────────
A: • 再次强制刷新浏览器 (Ctrl+Shift+R)
   • 清除浏览器缓存
   • 尝试无痕/隐私模式
   • 尝试不同浏览器

Q2: 动画不显示？
────────────────
A: • 检查浏览器控制台是否有错误
   • 确认浏览器版本（需 Chrome 76+）
   • 禁用 Cloudflare Rocket Loader
   • 验证 JavaScript 已加载

Q3: 某些用户说没更新？
──────────────────────
A: • 可能是 CDN 节点更新延迟
   • 等待 5-10 分钟
   • 或要求用户强制刷新

Q4: 如何检查是否使用了缓存？
──────────────────────────────
A: • F12 → Network → 刷新页面
   • 查看响应头 "CF-Cache-Status"
   • MISS = 未缓存（最新）
   • HIT = 使用缓存（可能旧）

┌───────────────────────────────────────────────────────────────────────┐
│  🎯 关键响应头检查                                                     │
└───────────────────────────────────────────────────────────────────────┘

在开发者工具中查看 styles.css 的响应头：

必须存在：
  ✓ CF-Cache-Status: MISS 或 DYNAMIC（第一次）
  ✓ Content-Type: text/css
  ✓ Last-Modified: 2024-03-07 (今天)

缓存控制：
  ✓ Cache-Control: max-age=...
  ✓ CF-RAY: ... (Cloudflare 标识)

Cloudflare 设置优化（可选）：

1. Auto Minify
   □ 开启 JavaScript 压缩
   □ 开启 CSS 压缩
   □ 开启 HTML 压缩

2. Brotli 压缩
   □ 开启 Brotli（比 Gzip 更好）

3. Early Hints
   □ 开启 HTTP/2 推送

4. Rocket Loader
   ⚠️ 如果动画有问题，禁用此项
   □ 或保持关闭状态

5. Page Rules
   建议设置不同文件类型的缓存时间：
   • HTML: 4 小时
   • CSS/JS: 1 周
   • 图片: 1 个月
